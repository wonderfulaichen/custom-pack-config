## 5分钟快速开始

### 1. 安装模组

1. 下载并安装 Minecraft Forge 1.20.1
2. 下载 Cloth Config 模组并放入 `mods` 文件夹
3. 下载本模组并放入 `mods` 文件夹
4. 启动游戏

### 2. 打开配置界面

有两种方式打开配置界面：

#### 方式一：主菜单按钮
- 打开游戏主菜单
- 点击"配置"（Config）或"数据包配置"按钮

#### 方式二：Mod Menu（如果已安装）
- 打开游戏主菜单
- 点击"模组"（Mods）
- 找到"Data Pack Config Mod"
- 点击配置按钮

### 3. 配置数据包

在配置界面中：
1. 查看已加载的数据包列表
2. 选择要配置的数据包
3. 修改配置项
4. 点击"完成"保存

## 常用配置类型

### 1. 布尔开关 (boolean)

**用途**: 开启/关闭功能

**示例**:
```json
{
  "enabled": {
    "type": "boolean",
    "default": "true",
    "name": {"zh_cn": "启用功能", "en_us": "Enable Feature"},
    "filePath": "data/example/config.json",
    "key": "enabled"
  }
}
```

### 2. 整数滑块 (slider)

**用途**: 调整整数值（0-100）

**示例**:
```json
{
  "spawn_rate": {
    "type": "slider",
    "default": "50",
    "min": 0,
    "max": 100,
    "step": 1,
    "name": {"zh_cn": "生成率", "en_us": "Spawn Rate"},
    "filePath": "data/example/config.json",
    "key": "spawn_rate"
  }
}
```

### 3. 浮点滑块 (double_slider)

**用途**: 调整浮点数值

**示例**:
```json
{
  "damage_multiplier": {
    "type": "double_slider",
    "default": "1.0",
    "min": 0.1,
    "max": 5.0,
    "step": 0.1,
    "name": {"zh_cn": "伤害倍率", "en_us": "Damage Multiplier"},
    "filePath": "data/example/config.json",
    "key": "damage_multiplier"
  }
}
```

### 4. 下拉菜单 (enum)

**用途**: 从选项中选择

**示例**:
```json
{
  "difficulty": {
    "type": "enum",
    "default": "normal",
    "enumValues": ["easy", "normal", "hard"],
    "name": {"zh_cn": "难度", "en_us": "Difficulty"},
    "enumDisplayNames": {
      "zh_cn": {"easy": "简单", "normal": "普通", "hard": "困难"},
      "en_us": {"easy": "Easy", "normal": "Normal", "hard": "Hard"}
    },
    "filePath": "data/example/config.json",
    "key": "difficulty"
  }
}
```

### 5. 颜色选择器 (color)

**用途**: 选择颜色

**示例**:
```json
{
  "theme_color": {
    "type": "color",
    "default": "#FF5733",
    "name": {"zh_cn": "主题颜色", "en_us": "Theme Color"},
    "filePath": "data/example/config.json",
    "key": "theme_color"
  }
}
```

## 创建你的第一个配置文件

### 步骤 1: 创建数据包文件夹

```
my_datapack/
├── pack.mcmeta
├── datapack_config.json
└── data/
    └── mynamespace/
        └── config.json
```

### 步骤 2: 创建 pack.mcmeta

```json
{
  "pack": {
    "pack_format": 15,
    "description": "我的数据包"
  }
}
```

### 步骤 3: 创建配置文件 (config.json)

```json
{
  "enabled": true,
  "difficulty": "normal"
}
```

### 步骤 4: 创建 datapack_config.json

```json
{
  "metadata": {
    "name": {
      "zh_cn": "我的数据包",
      "en_us": "My Datapack"
    },
    "description": {
      "zh_cn": "我的第一个数据包配置",
      "en_us": "My first datapack config"
    }
  },
  "config": {
    "enabled": {
      "type": "boolean",
      "default": "true",
      "name": {
        "zh_cn": "启用数据包",
        "en_us": "Enable Datapack"
      },
      "filePath": "data/mynamespace/config.json",
      "key": "enabled"
    },
    "difficulty": {
      "type": "enum",
      "default": "normal",
      "enumValues": ["easy", "normal", "hard"],
      "name": {
        "zh_cn": "难度",
        "en_us": "Difficulty"
      },
      "enumDisplayNames": {
        "zh_cn": {
          "easy": "简单",
          "normal": "普通",
          "hard": "困难"
        },
        "en_us": {
          "easy": "Easy",
          "normal": "Normal",
          "hard": "Hard"
        }
      },
      "filePath": "data/mynamespace/config.json",
      "key": "difficulty"
    }
  }
}
```

### 步骤 5: 安装数据包

1. 将 `my_datapack` 文件夹复制到数据包配置模组的数据包文件夹
   - 默认路径: `config/datapack-config-mod/my_datapack`
2. 启动游戏
3. 打开配置界面
4. 查看并修改配置

## 常见问题

### Q: 配置修改后没有生效？

A: 确认以下几点：
1. 数据包已重新加载（通常需要重新进入世界）
2. 检查 `datapack_config.json` 中的 `filePath` 是否正确
3. 查看游戏日志确认配置是否成功应用

### Q: 配置界面显示不正常？

A: 检查以下几点：
1. JSON 格式是否正确（使用 JSON 验证工具）
2. 所有必需参数都已配置
3. 文件编码是否为 UTF-8

### Q: 如何批量修改多个文件？

A: 使用 `locations` 数组：

```json
{
  "debug_mode": {
    "type": "boolean",
    "default": "false",
    "locations": [
      {
        "filePath": "data/example/config1.json",
        "key": "debug",
        "description": "配置1"
      },
      {
        "filePath": "data/example/config2.json",
        "key": "debug",
        "description": "配置2"
      }
    ]
  }
}
```

### Q: 如何修改嵌套的 JSON 对象？

A: 使用 `/` 分隔路径：

```json
{
  "terrain_scale": {
    "type": "double_slider",
    "default": "1.0",
    "filePath": "data/example/terrain.json",
    "key": "terrain/scale"
  }
}
```

对应文件内容：
```json
{
  "terrain": {
    "scale": 1.0
  }
}
```

### Q: 如何添加本地化？

A: 在配置项中添加多语言文本：

```json
{
  "difficulty": {
    "name": {
      "zh_cn": "难度",
      "en_us": "Difficulty"
    },
    "description": {
      "zh_cn": "设置游戏难度",
      "en_us": "Set game difficulty"
    },
    "tooltip": {
      "zh_cn": "点击选择难度",
      "en_us": "Click to select difficulty"
    }
  }
}
```
